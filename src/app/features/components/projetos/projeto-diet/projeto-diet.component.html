Projeto Api Diet Rocketseat

Abaixo uma breve descrição e explicação do projeto além da apresentação das tecnologias utilizadas para desenvolvimento para esse meu projeto pessoal.

Vuex

TypeScript

Node.js

JavaScript

Este projeto foi desenvolvido como parte de um desafio da plataforma de cursos Rocketseat, com o objetivo de criar uma API robusta utilizando Node.js com TypeScript e o framework Fastify, substituindo o uso do Express. O sistema foi desenhado para gerenciar as dietas dos usuários através de operações CRUD (Criar, Ler, Atualizar, Deletar).
Originalmente proposto para ser construído em React, o projeto foi reestruturado utilizando Vuex para gerenciamento de estado, com HTML e CSS puro para uma interface responsiva.

FUNCIONALIDADES

Registro de Refeições

Os usuários podem registrar cada refeição, incluindo detalhes como horário e data.

Edição de Dados

Após o registro, os usuários têm a liberdade de editar qualquer informação sobre suas refeições.

Análise Gráfica

O sistema gera gráficos que analisam o equilíbrio das dietas, destacando refeições saudáveis e menos saudáveis.

Exclusão de Refeições

Os usuários podem excluir registros de refeições, permitindo um controle detalhado sobre a dieta.
